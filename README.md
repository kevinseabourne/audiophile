# E-commerce Website

This e-commerce website allows you to purchase headphones, speakers or earphones. You can add these products to your cart, complete the checkout process and your order will be hypothetically on its way.

## Stack

• Next.js
• TypeScript
• Styled Components
• Firebase
• Framer Motion
• React Hook Form
• Big.js
• Accounting.js

## Features

• HTTP Requests
• Cart Functionality
• Checkout Process
• Dynamic Routes
• Responsive
• Accessibility
